Menu digital QR Code pour votre activité

Offrez une expérience sans contact à vos clients et faites gagner du temps à vos serveurs grâce à un menu digital sur smartphone à scanner avec un QR Code. MY QRCODE SOLUTIONS vous sert de générateur de QR code pour vos cartes de restaurant à flasher.

COMMENT ÇA MARCHE ?

Pour les clients


1. Ouvrez l'application appareil photo sur votre smartphone.
2. Dirigez la caméra vers le code QR.
3. Assurez-vous que le code se trouve dans le champ de vision de la caméra et qu'il est bien éclairé.
4. Attendez que le code soit reconnu par la caméra.
5. Une notification ou un message peut apparaître sur votre écran, indiquant que le code a été scanné.
6. Suivez les instructions fournies par la notification ou le message.
7. Cela inclura l'ouverture d'un site Web qui affichera le menu
